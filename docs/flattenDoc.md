##Flatten

###`flatten(array)`

Takes one level of nested array and inserts it in place into the parent array.

###Arguments

array (Array): The array to flatten.

###Returns

(Array): The flattened array.

###Example

```javascript
bland.flatten([[1,2],[5,3,[4,6]],7])
//=> [1,2,5,3,[4,6],7]

//=>
```
