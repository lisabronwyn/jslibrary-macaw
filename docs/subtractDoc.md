##Subtract

###`subtract(value1, value2)`

Subtracts value2 from value1.

###Arguments

value1 (number): The first value.

value2 (number): The value to subtract.

###Returns

(numbers): Returns the difference between the two values.

###Example

```javascript
bland.subtract(5,2)
//=> 3

bland.subtract(6,10)
//=> -4
```
